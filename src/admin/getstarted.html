Getting Started

  1. Open /src/db/config/dbconfig.php in a text editor and ensure that all the details are correct.
  2. Open /src/db/config/tablesconfig.php in a text editor and ensure that all the details are correct.
  3. Open /src/db/controllers/tablesetup.php in a text editor and verify that all data types are correct.
  4. Open /src/public/docs/privacy.html (Privacy Policy) in a text editor and ensure that all information is correct.
  5. Open /src/public/docs/tos.html (Terms of Service) in a text editor and ensure that all information is correct.
  6. Open /src/public/docs/faq.html (Frequently Asked Questions) in a text editor and add more information if necessary.
  7. Open /src/public/docs/changelog.html (Changelog) in a text editor and display any changes made to this application.
  8. Open /src/db/setup/dbsetup.php in a web browser to finish setting up this application.

Optional changes:

  • Delete setup directory in /src/db/

And last, but not least, don't forget to customize your site!
