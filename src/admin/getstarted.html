Getting Started

  1. Open /src/db/dbconfig.php in a text editor and ensure that all the details are correct.
  2. Open /src/db/tablesconfig.php in a text editor and ensure that all the details are correct.
  3. Open /src/db/tablesetup.php in a text editor and verify that all data types are correct.
  4. Open /src/docs/privacy.html (Privacy Policy) in a text editor and ensure that all information is correct.
  5. Open /src/docs/tos.html (Terms of Service) in a text editor and ensure that all information is correct.
  6. Open /src/docs/faq.html (Frequently Asked Questions) in a text editor and add more information if necessary.
  7. Open /src/docs/changelog.html (Changelog) in a text editor and display any changes made to this application.
  8. Open /src/docs/dbsetup.php in a web browser to finish setting up this application.

Optional changes:

  • Delete /src/db/dbsetup.php

And last, but not least, don't forget to customize your site!
